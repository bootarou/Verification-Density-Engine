# Verification Density (VD) Engine

BEYONDで使用する「データ評価エンジン」です。  
検証履歴をもとに、データの信頼性を数値化します。

---

## ✨ 概要

このライブラリは、以下を計算します：

- **Verification Density（VD）**
  - データがどれだけ検証されているか
- **Trust Score**
  - データがどれだけ安定して信頼できるか

---

## 🧠 コンセプト

従来：
データ = 真実（とみなす）

BEYOND：
データ + 検証履歴 = 信頼性

さらに、

誤りは価値を下げるのではなく、検証プロセスとして価値になる

---

## 📦 インストール

```bash
# 任意のプロジェクトにコピー
calculateVD.ts
```

---

## 🚀 使い方

```ts
import { calculateVD } from "./calculateVD";

const result = calculateVD({
  Nc: 12,
  Ne: 4,
  Ncorr: 3,
  Napp: 6,
  Na: 5,
  U: 1,
  L: 1800,
  F: 14,
  C: 8
});

console.log(result);
```

## 🧮 計算式

VD(R,t) =
(w1Nc + w2Ne + w3Ncorr + w4Napp + w5Na - w6U)
------------------------------------------------
(αL + βF + γC)

---

## 🔢 パラメータ説明

### 検証指標

| パラメータ | 説明 |
|----------|------|
| Nc | 一致確認 |
| Ne | 誤り指摘 |
| Ncorr | 訂正 |
| Napp | 追記 |
| Na | 承認 |
| U | 未解決誤り |

---

### スケール指標

| パラメータ | 説明 |
|----------|------|
| L | データ長 |
| F | フィールド数 |
| C | 主張数 |

## 📈 Trust Score

Trustは以下で計算されます：

- 誤り解決率
- 承認・訂正・一致確認

---

## 🖥 UI表示例

### VD

| 状態 | 条件 |
|------|------|
| Unverified | vd < 0.45 |
| In Review | 0.45 <= vd < 0.85 |
| Verified | vd >= 0.85 |
| Contested | 未解決誤りが多い |

---

### Trust

| 状態 | 条件 |
|------|------|
| Low | < 0.4 |
| Medium | 0.4 - 0.7 |
| High | >= 0.7 |

---

## 🔁 データフロー

作成
 ↓
一致確認 (Nc)
 ↓
誤り指摘 (Ne)
 ↓
未解決 (U)
 ↓
訂正 (Ncorr)
 ↓
追記 (Napp)
 ↓
承認 (Na)
 ↓
VD再計算

---

## ⚠️ 重要な設計原則

### ❌ 上書き禁止
履歴が消えるため評価不能になります

### ✅ 訂正と追記を分離
意味が全く異なります

### ✅ 未解決誤りを必ず管理
評価の歪みを防ぐため
